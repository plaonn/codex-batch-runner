# 로컬 Tasks 템플릿

이 파일은 개인 운영 환경에서 사용할 작업 대시보드 템플릿입니다. 실제 메모는 저장소 루트의 `TASKS.local.md`로 복사해서 사용하십시오. `TASKS.local.md`는 gitignore 대상이어야 하며, 공개 저장소에 커밋하지 않습니다.

`TASKS.local.md`에는 현재 판단과 실행을 돕는 짧은 작업 목록만 기록합니다. 상세 설계 본문은 `docs/spec.md`에서 연결되는 topic 문서 또는 별도 제안 문서에 두고, 완료 로그는 Git 기록과 커밋에 맡깁니다.
