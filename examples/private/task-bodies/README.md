# Private Task Bodies Template

이 디렉터리는 `.private/TASKS.md`의 큰 active task 본문을 두는 템플릿입니다.

실제 개인 운영 환경에서는 저장소 루트의 `.private/task-bodies/` 아래에 복사해서 사용하십시오. task body는 장기 archive가 아니며, 연결된 `TASKS.md` 항목이 완료, 취소, supersede되어 삭제되면 함께 삭제합니다.

task body에는 설계 메모, 분해 계획, acceptance 기준, 열린 질문처럼 해당 task를 수행하는 데 필요한 내용만 둡니다.
